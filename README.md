### Sobre:	
#### Mountain Shooter é um arcade game 2D desenvolvido com Pygame.	
#### O jogo tem 3 níveis, e cada nível termina com timeout. 
#### Pode ser jogado de forma individual ou cooperativa. 
#### A pontuação é salva no SQLite3.
